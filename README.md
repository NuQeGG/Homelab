# Homelab

Welcome to my homelab repository — a place where I document my infrastructure, experiments, services, and lessons learned while building and maintaining my own self-hosted environment.

## What this repo contains

This repository is used to keep track of:

* Hardware and server setup
* Network layout (Under planning)
* Virtual machines and containers
* Services I run
  
## Why I made this

The goal is to document everything clearly so I can:

* rebuild faster if something breaks
* learn infrastructure properly
* keep configurations organized
* track progress over time
